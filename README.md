# Modulebank test job
This is the test job for hiring to **[Modulbank](https://modulbank.ru)**.
**[DEMO](http://***.now.sh)**

## Install
```
git clone git@github.com:maksimdegtyarev/modulebank-test.git
cd modulebank-test
```
then
```
npm install
```
or
```
yarn install
```
then
```javascript
npm run-script build
```
then open ./dist/index.html in your favourite browser.
